## 📌 Description

**Glass Auth Template** is a ready-to-use authentication template with modern glassmorphism design. It includes a complete authentication system powered by **Supabase**, 3 color themes, and support for 10 languages.

### Who is this template for?

| Category | Description |
|----------|-------------|
| 🚀 **Startups** | Quick start for web applications |
| 👨‍💼 **Dashboards** | Beautiful and functional login |
| 🛠️ **Admin Panels** | Modern design for management |
| 📁 **Portfolio** | Showcase your frontend skills |

---

## ✨ Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | 🪟 **Glass Design** | Modern Glassmorphism with blur and transparency |
| 2 | 🌙 **3 Themes** | Dark, Light, Color (with gradient) |
| 3 | 🌍 **10 Languages** | Multi-language support |
| 4 | 📱 **Responsive** | Works perfectly on all devices |
| 5 | 🔐 **Authentication** | Login and registration via Supabase |
| 6 | 🎨 **Color Customization** | Change background, glass color and opacity |
| 7 | ✨ **Smooth Animations** | Pleasant micro-animations on load |
| 8 | 🖱️ **Interactivity** | Hover effects on all elements |
| 9 | 🛡️ **Security** | Protected against autofill with white background |
| 10 | 🔄 **Settings Persistence** | All settings saved in localStorage |

## 📌 Описание

**Glass Auth Template** — это готовый шаблон страниц входа, регистрации и профиля со стеклянным дизайном (Glassmorphism). Шаблон включает систему авторизации на базе **Supabase**, 3 цветовые темы и поддержку 10 языков.

### Для кого этот шаблон?

| Категория | Описание |
|-----------|----------|
| 🚀 **Стартапы** | Быстрый старт для веб-приложений |
| 👨‍💼 **Личные кабинеты** | Красивый и функциональный вход |
| 🛠️ **Админ-панели** | Современный дизайн для управления |
| 📁 **Портфолио** | Демонстрация навыков |
